Phase P1.2.1 — Observability Settings Screen

Added an Admin-only Performance & Observability screen inside Settings & Permissions.
The screen displays startup, network, slow-request, JavaScript error, active-screen, and memory metrics with refresh and JSON export actions.
Changed the optional shortcut from Ctrl+Shift+P to Ctrl+Shift+O to avoid the browser Print conflict.
No business logic, Supabase schema, payroll, commission, security, or permission rules were changed.

Version: PETATOE v10.0.4
