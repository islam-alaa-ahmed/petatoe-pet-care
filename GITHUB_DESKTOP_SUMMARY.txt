PETATOE v9.4.14 — Smart Reports Data Readiness Recovery

- Kept render coalescing, localization batching, and duplicate-render protection from v9.4.13.
- Added Smart Reports data-readiness recovery validation to GitHub Actions.
- Synchronized release metadata and cache tokens to PETATOE v9.4.14.
